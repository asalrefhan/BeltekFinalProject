# BeltekFinalProject
Final Project for Beltek C# ASP.Net Course

Models: 
Ogrenci, Ogretmen

DbContext: 
OkulDbContext

Controllers: 
OgrenciController, OgretmenController, HomeController

Ogrenci-> OgrenciListele, OgrenciEkle, OgrenciDuzenle
Ogretmen-> OgretmenListele, OgretmenEkle, OgretmenDuzenle
